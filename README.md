# AI Futuristic Product Lab

Laboratório de estudos e desenvolvimento focado em IA, sites futuristas, SaaS, dashboards, agentes e produtos digitais.

## Objetivo

Construir uma trilha prática de estudos com projetos reais, usando IA em todas as etapas do processo: pesquisa, design, desenvolvimento, revisão, testes, documentação e deploy.

## Projetos

- 01 - NeuralOps AI
- 02 - Orbit Studio
- 03 - Chronos Interface
- 04 - Quantum Landing
- 05 - InsightOS AI
- 06 - AgentDesk AI
- 07 - AffiliateVision AI

## Método

Cada projeto deve conter:

- PRD
- Protótipo ou referência visual
- Código versionado
- README próprio
- Deploy
- Prints
- Registro de aprendizados
